public enum ResultIdentifier : String, CodingKey

The result identifiers used for the additional results recorded by the distance recorder.

  • stepCount

    The step count returned by the pedometer.

  • gpsDistance

    The total distance measured by summing the relative distance travelled during the motion step (when the participant is expected to be moving).

    Note

    it is possible to game this result because the calculation of this result does not account for coordinate measurement accuracy or the course and speed at which the user is travelling. It is used only to give feedback to a partipant. The pedometer distance and a path constructed by using the course, speed, and relative distance recorded samples should be used by a research study for more accurate data collection.
